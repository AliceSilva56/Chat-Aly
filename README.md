# Chat-Aly

Nome do projeto: Chat Aly

Descrição: O chat Aly, é um chat de conversas programadas, tem também uma história interativa
O chat Aly vai conter 

- Uma tela início, onde vai ter a caixinha de mensagem da Aly, quando for clicado vai direcionar para o chat 

NO CHAT

- Já vai ter um uma mensagem do chat
- Caixa de entrada de texto para envio das mensagens
- Nos 3 pontinhos vai poder trocar o background do chat
- clicando no nome Aly, você abre uma descrição do contato, onde você pode até ligar
